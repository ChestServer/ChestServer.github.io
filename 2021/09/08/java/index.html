关于配置Java环境变量的教程

发表于2021-09-08|更新于2021-09-08

相关文件在群里可以找到！
众所周知我的世界是使用java开发的，需要先配置好Java环境变量才能够运行

所以这是一篇关于配置Java环境变量的教程

教程正文

进入环境变量编辑

桌面右键此电脑，选择属性，进入高级系统设置

选择环境变量

JAVA_HOME变量

在系统变量（或者用户变量）中新建变量
JAVA_HOME
变量值为Java安装路径，比如此处我的路径为：
C:\Program Files\Java\jdk1.8.0_212\bin

Path变量

新建或编辑变量
Path
新建变量值
C:\Program Files\Java\jdk1.8.0_212\bin

CLASSPATH变量

新建变量
CLASSPATH
变量值
.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

至此环境变量的配置结束，接下来验证Java运行环境

Java运行环境验证

windows+R键打开运行，输入cmd，点击确定

输入
javac
回车

输入
java -version
回车

若都出现对应内容，即证明环境变量设置成功
